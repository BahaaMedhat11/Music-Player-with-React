# 🎵 Music Player

A modern and responsive Music Player built with React that allows users to browse songs, play audio tracks, create custom playlists, and manage their music experience with an intuitive interface.

## ✨ Features

- 🎧 Play and pause audio tracks
- ⏭ Skip to next track
- ⏮ Return to previous track
- 📊 Interactive progress bar
- 🔊 Volume control
- 🎵 Browse all available songs
- 📁 Create custom playlists
- ➕ Add songs to playlists
- 🗑 Delete playlists
- 💾 Persistent playlists using Local Storage
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/music-player.git
```

### Navigate to Project

```bash
cd music-player
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

---

## 🎮 Usage

### Play Songs

- Open the application
- Select any song from the Songs page
- Click Play to start listening

### Create Playlist

- Navigate to Playlists
- Enter a playlist name
- Click Create

### Add Songs

- Search for songs inside a playlist
- Select a song from the dropdown
- The song will be added instantly

### Delete Playlist

- Click the Delete button
- Confirm the action
